关于 Markdown 的使用

Markdow 的优点

  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开
  • 专注于文字写作而不是排版
  • 格式转换方便，可以轻松转换为 HTML、PPT、电子书等
  • Markdown 标记语法简单好记，可读性强
