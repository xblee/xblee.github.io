命令行

Posted on 2014-08-27

命令行

shell

shell 就是一个程序，接收键盘输入的命令，传给操作系统去执行

几乎所有的 Linux 都提供了一个名为 bash 的 shell 程序

终端仿真器

当使用图形用户界面时，我们需要另一个和 shell 交互的叫做终端仿真器的程序。 在 Windows 上，一般使用操作系统自带的 cmd 或者 powershell。 在 Linux 上，如果是图形用户界面，那么可以使用 terminal 或者 konsole、gnome-terminal 之类 的终端仿真器，但基本上，它们都完成同样的事情，让我们能访问 shell， 你可能会因为附加的一些花俏功能而喜欢上某个终端。

关于名字，如果有人提到：控制台、终端、bash、shell、terminal 等，一般都是指上面这些。

一般在 Linux 上，都会提供一个 bash 或者 terminal 之类的工具，让你来使用 shell

安装 Git for Windows 客户端体验 shell

Git for Windows 客户端自带了一个终端仿真器：git bash

[下载地址：]（https://git-for-windows.github.io/）

git for Windows - github 托管地址 ，然后找到 Realease 模块，找到下载的对应版本即可。

注意：虽然在 Windows 上可以通过 cmd 或 powershell 体验终端的使用， 但是学习终端最正确的方式是学习 Linux 上的命令行，要更纯正，更强大。

所以，我们这里通过使用 Git for Windows 客户端（自带了一个 bash）可以供我们学习 shell。

打开和退出终端

命令提示符

git bash 中使用 ~ 来表示用户目录

使用终端操作文件系统

  • 绝对路径和相对路径
  • pwd （print working directory）
    • 查看当前路径
  • cd （change directory）
    • 切换到指定路径
  • ls （list files）
    • ls
      • 列出当前目录文件
    • ls 目录路径
      • 列出指定路径文件
    • ls -a
      • 列出文件并显示隐藏文件
  • cp （copy）
    • cp 源路径 目标路径
    • cp 在复制目录的时候，不会复制里面的子目录和子文件
    • cp -r 递归复制（深度复制）
  • mv （move）
    • 移动文件或目录，也可以重命名
  • mkdir （make directory）
    • 创建目录
  • rm （remove）
    • 删除
    • -r 递归删除
  • rmdir 目录名称
    • 删除指定空目录
  • clear 清屏
  • touch 文件名
    • 创建该文件
  • cat 文件名
    • 查看指定的文本文件

使用 less 命令浏览文件内容

less 命令是一个用来浏览文本文件的程序

less 文本文件名

在 less 中使用 q 退出 less 程序

文本编辑器 vi

vi 全称 visual interface

类似于 windows 中的记事本 notepad

但比 notepad 强大很多

vi 操作

  • vi 启动vi
  • :q 退出vi
  • :q! 强制退出
  • esc 按两次可以退回普通模式
  • vi 文件名 创建新文件
  • vi 文件名 编辑已有文件
  • i 进入插入模式 insert 后才能编辑文本内容
    • 输入完成后按 ESC 退出插入模式到命令模式

保存工作

  • :w 保存
  • :wq 保存并退出
  • ZZ 保存并退出
  • :w 文件名 指定可选文件名 类似于另存为

基本编辑

插入

  • a 越过行尾，进入插入模式
  • A 直接移动到当前行末尾，进入插入模式
  • o 在当前行的下方打开一行，进入插入模式
  • O 在当前行的上方打开一行，进入插入模式

撤销

  • u 撤销最后一次修改
  • ctrl + r 回复撒谎那个一步被撤销的操作

删除

  • x 删除当前字符
  • 3x 删除当前字符及气候的两个字符
  • dd 删除当前行
  • 5dd 当前行及随后的四行文本
  • dW 从光标开始位置到下一个单词的开头
  • d$ 从光标位置开始到当前行的行尾。
  • d0 从光标位置开始到当前行的行首。
  • dˆ 从光标位置开始到文本行的第一个非空字符。
  • dG 从当前行到文件的末尾。
  • d20G 从当前行到文件的第 20 行

复制

  • yy 当前行。
  • 5yy 当前行及随后的四行文本。
  • yW 从当前光标位置到下一个单词的开头。
  • y$ 从当前光标位置到当前行的末尾。
  • y0 从当前光标位置到行首。
  • yˆ 从当前光标位置到文本行的第一个非空字符。
  • yG 从当前行到文件末尾。
  • y20G 从当前行到文件的第 20 行。
  • J 把行与行之间连接起来

粘贴

  • p 将剪切板中的文本粘贴到光标位置之后
  • P 将文本粘贴到光标之前

查找替换

  • f 要查找的字符
    • 通过输入分号查找当前行下一个
  • / 要查找的字符
    • 输入 n 查找下一个
