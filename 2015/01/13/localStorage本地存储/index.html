localStorage本地存储

Posted on 2015-01-13

localStorage

本地存储

当作一个仓库

有三个重要方法

setItem

window.localStorage.setItem(“user”,JSON.stringify([{“username”:”admin”,”password”:”123456”},{“username”:”admin”,”password”:”123456”},{“username”:”admin”,”password”:”123456”}]))

存储进去后，将对象转为了字符串

所以这里要进行 json 序列化：通过 JSON.stringify 方法，将 json 对象转换成 json 字符串

（另外还有 json 反序列化：将 json 字符串转换为 json 对象）

getItem

window.localStorage.getItem(“user”)

得到了”[{“username”:”admin”,”password”:”123456”},{“username”:”admin”,”password”:”123456”},{“username”:”admin”,”password”:”123456”}]”

可以通过 json.parse 解析一下

JSON.parse(window.localStorage.setItem(“user”))

得到 [object,ovject,object]

而第一次用户没有输入用户名、密码的时候

JSON.parse(window.localStorage.setItem(“user”))的结果为 null

所以要给这个数组一个初始值

JSON.parse(window.localStorage.setItem(“user”)||”[]”)
