数组的 forEach 方法

ES5 新增的数组的 forEach 方法
用来遍历数组
比如

等同于

该方法传入的参数为

[].forEach(function(value,index,array){})

另外forEach循环不能退出，匹配到了还会继续循环，有损性能

对比 jQuery 中的 $.each 方法

$.each([],function(index,value,array){})

他们的传参顺序是不同的

类似的还有 jQuery 中的 map 方法

forEach 方法的特点

forEach 循环不能退出，即使匹配到了也会把后续的循环执行完

这里不能用 break 和 return 来打断

注意： 没有办法中止或者跳出 forEach 循环，除了抛出一个异常。如果你需要这样，使用forEach()方法是错误的，你可以用一个简单的循环作为替代。如果您正在测试一个数组里的元素是否符合某条件，且需要返回一个布尔值，那么可使用 Array.every 或 Array.some。
some 只要一个返回 true 则遍历结束
every 所有都要为 true
