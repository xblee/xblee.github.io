AJAX

同步交互

客户端给服务器请求，服务器给客户端响应
响应回来的页面会把原来的页面覆盖掉
服务器给客户端进行处理，处理过程中，用户只能等待，不能操作

异步交互

客户端给服务器请求，服务器给客户端响应
响应回来的页面不会把原来的页面覆盖掉
用户可以进行其他操作

XMLHttpRequest

js提供了一个构造函数——XMLHttpRequest
它可以用来发送请求和接收响应
数据返回后，它可以把数据写到页面上
通过xml可以发送请求首行、请求头、请求体
通过xml可以接收响应首行、响应头、响应体

通过XMLHttpRequest和服务器的交互

常见200表示成功，403没有权限，404未找到，500服务器错误
304走的是缓存，也可以认为是请求成功了

get提交

创建对象    var    xhr = new XMLHttpRequest（）
跟服务器连接    xhr.open("GET","get.php?username = lixiaobo")
发送数据    xhr.send(null)    //get方式提交不发送数据，但是必须写null
接收数据    编写一个回调函数    //每当 readyState 属性改变时，就会调用该函数
        xhr.onreadystatechange = function (){
            //xhr.readyState 是服务器的返回状态 0,1,2,3,4 其中4代表服务器响应完毕
            //xhr.status 是获取服务器状态，200为响应成功，404为请求资源未找到，500为服务器内部出错

            if(xhr.readyState == 4 && xhr.status == 200){
                var data = xhr.responseText
            }
        }
