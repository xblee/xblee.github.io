关联数组

关联数组的数据是有关联的，通过键值对相关联
$arr = array(“username”=>”boblee”,”age”=>”18”);

遍历关联数组

foreach（$array as $键）{

}
foreach（$array as $键=>$值）{
echo $键;
echo $值;
}
